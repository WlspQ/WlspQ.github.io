Hacknos-3 靶机

kali ip：192.168.1.67

target ip：192.168.1.155

靶机下载：https://www.vulnhub.com/entry/hacknos-os-hacknos-3,410/

靶机目标：普通用户的 user.txt 和 root 用户的 root.txt

使用 nmap 进行网络扫描，发现目标 192.168.1.155 为入侵目标靶机…

访问 192.168.1.155

使用 dirsearch 进行目录扫描

发现路径：http://192.168.1.155/scripts/

先访问 http://192.168.1.155/scripts/ 发现第一个点击后返回网页根目录，除了 automail.pl 能访问外，其它的都不能访问，并且 automail.pl 里面没有比较有用的信息

发现 Websec 并访问得出关键信息 contact@hacknos.com

按照传统习惯，访问 admin，发现一个后台登录，用发现的 contact@hacknos.com 为用户名

在 http://192.168.1.204/websec/ 中获取可能的密码口令（网络原因靶机 IP 变为 192.168.1.204）

使用 hydra 爆破密码

获得密码为 Securityx

使用该密码登录进入后台，发现文件上传点

进入后台后尝试 GetShell，在 Content>>Media 处可以上传图片， Content>>File Manager 处可以编辑 PHP 文件… 在 Content>>File Manager 处进入 tmp 目录编辑 .htaccess 文件，将里面的内容全部删除点击 Save 并进入 tmp/media_thumb/ 目录下上传一句话文件，访问并使用蚁剑进行链接…

使用蚁剑进行链接并上传由 msf 生成的 Shell 文件

在 kali 中启动 msfconsole 并进去到监听模式下… 并打开浏览器访问 shell.php 进行 shell 反弹…

在 meterpreter 下输入 shell 命令进入目标 shell 环境并执行以下命令进入 bash 环境…

编译以下程序源码并上传到靶机目录下，给于 777 权限并执行获取高权限… 并拿去 Flag…
